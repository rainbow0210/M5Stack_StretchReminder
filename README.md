# M5Stack_StretchReminder

## 概要

長時間のデスク作業は、肩こりや腰痛などの現代人が抱える一般的な健康問題の主要な原因です。この「座業問題」に着目し、適切なタイミングでユーザーに休憩とストレッチを促すスマートリマインダーを開発しました。本デバイスはユーザーの健康維持を支援し、より効果的に体をリフレッシュできることを目指しています。

## 使用技術

- 言語: C++ (Arduino)
- 開発環境: Arduino IDE
- ハードウェア: M5Stack Core2 for AWS
- センサ: 超音波距離センサー (M5Stack Ultrasonic Distance Unit I2C)、6軸IMUジャイロセンサー
- ライブラリ: M5Unified、M5Unit-Sonic
- 接続: Grove ケーブル
- AI補助: Gemini 2.5 Pro を使用したコーディング支援

## 使い方

### 前提条件

**ハードウェア:**
- M5Stack Core2 for AWS
- M5Stack Ultrasonic Distance Unit I2C

**開発環境:**
- Arduino IDE

### インストール方法

1. Arduino IDE の Library Manager を開き、以下の2つのライブラリをインストールしてください:
   - M5Unified
   - M5Unit-Sonic (by M5Stack)

2. M5Stack Ultrasonic Distance Unit I2C を Grove ケーブルを使用して M5Stack Core2 for AWS の PORT.A に接続してください。

3. 本リポジトリのコードを Arduino IDE で M5Stack Core2 for AWS にアップロードしてください。

### 基本的な使い方

**デスク作業リマインダーモード:**
- デバイスをデスクの前に配置してください
- 超音波センサーがデスク前の姿勢を検出します
- 設定時間（デフォルト30分）以上の作業を検出するとアラートが鳴ります
- ボタンを押してアラートを停止してください

**姿勢リマインダーモード:**
- デバイスを身体に装着してください
- ジャイロセンサーがリアルタイムで身体の傾きを検出します
- 設定時間（デフォルト30分）以上同じ姿勢が続くとアラートが鳴ります
- ボタンを押してアラートを停止してください

## 主な機能

1. **デスク作業リマインダー**
   - 超音波距離センサーでデスク前の作業時間を測定
   - 設定時間を超えると音声アラートを発動
   - 長時間の連続作業を防止し、休憩を促進

2. **姿勢リマインダー**
   - M5Stack 内臓のジャイロセンサーで身体の傾きを検出
   - 同じ姿勢が長く続くと音声アラートを発動
   - 姿勢変化を促進し、血流改善をサポート

3. **簡単操作**
   - ボタンワンプレスでモード切り替え
   - ボタンワンプレスでアラート停止

4. **カスタマイズ可能な設定**
   - リマインダー間隔はコード内で変更可能（デフォルト30分）
   - 音量調整機能

## 設定

リマインダー間隔はコード内の以下の行で変更できます:

```cpp
unsigned long reminder_duration_ms = 30 * 60 * 1000; // ミリ秒単位で設定
```

例: 15分に設定する場合
```cpp
unsigned long reminder_duration_ms = 15 * 60 * 1000;
```

音量設定も同様にコード内で調整可能です。
